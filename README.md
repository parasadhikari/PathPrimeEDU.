# 🎓 PathPrimeEdu

> **An AI-powered EdTech platform built to simplify learning, coaching management, and student engagement.**

PathPrimeEdu is a full-stack EdTech platform designed for coaching institutes, teachers, and students. It provides a centralized system for course management, study materials, previous year questions, admissions, community discussions, fee management, and much more. The platform is also available as an Android application using Capacitor.

🌐 **Website:** https://pathprimeedu-frontend.onrender.com

## 📖 Project Overview

PathPrimeEdu is designed to digitize coaching institutes by providing a modern learning management platform.

It enables:

- 👨‍🏫 Teachers/Admins to manage students and study materials.
- 👨‍🎓 Students to access notes, PYQs, courses, announcements, and community discussions.
- 📱 Cross-platform accessibility through both Web and Android.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/parasadhikari/PathPrimeEDU.git
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

# 🔐 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=

MONGO_URI=

JWT_SECRET=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=

EMAIL_USER=

EMAIL_PASS=
```
